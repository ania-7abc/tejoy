# Contributing

We welcome contributions! You can open an issue or submit a pull request

## Issues

Before creating a new issue:

1. Check the [FAQ](https://ania-7abc.github.io/tejoy/faq) to see if your question is already answered
2. Search the existing issues to avoid duplicates
3. If you still need to open an issue, please include:
   - A minimal reproducible example - avoid posting large code blocks (100+ lines)
   - A detailed description of the problem, including expected vs. actual behavior

## Pull Requests

### License
All code is licensed under the MIT License. By submitting a pull request, you automatically agree to license your contributions under the same terms

#### Modifying an existing file
On copyright notice, add "and others" to the copyright line  
For example, if the original is:

```cpp
Copyright (c) 2026 Original Author
```

change it to:

```cpp
Copyright (c) 2026 Original Author and others
```

#### Creating a new file
Place the following template at the top of the file:

```cpp
/**
 * @file filename
 * @brief brief description
 * @copyright Copyright (c) 2026 Your_GitHub_username_or_alias SPDX-License-Identifier: MIT
 */
```

### Code Style
- Follow the existing coding style of the project
- Write Doxygen comments

Thank you for your contribution!
